# Card players - React App

A simple React application that displays players in cards and allows for dynamic filtering in real time.  
Built as a practice project to improve project structure, component organization, React, SASS, and UI/UX skills.

## Technologies

* HTML
* SCSS
* JavaScript
* React
* Font Awesome
* Vite
* ESLint
